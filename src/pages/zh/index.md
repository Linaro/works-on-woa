欢迎使用适用于 Arm 架构的 Windows 操作系统！

本网站为运行在 Arm 架构上的 Windows 设备提供应用和游戏的兼容性信息，并会持续更新。

若想了解我们如何定义兼容性以及如何使用本网站，请查看 [帮助页面](/zh/help)。

如果您已经测试过应用或游戏，并希望分享您的测试结果，请参考投 [稿指南](/zh/contributing)，了解如何通过 GitHub 向开源存储库提交拉取请求。

请注意，本站提供的信息仅供参考，无法完全保证某款游戏能够正常运行。尽管测试结果已验证，但在您的具体设备和配置上可能会有所不同。如果您的测试结果有所不同，欢迎向本网站分享您的发现。
